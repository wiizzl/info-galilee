# 2/ Définitions

## Réseau local

Un réseau local est un ensemble de machines reliées entre elles. Pour que ces machines puissent communiquer entre elles, il faut les connecter grâce soit à un support filaire (cable RJ45 par exemple) ou soit par un support aérien de type wifi.

Dans le cas d'un réseau filaire, on connecte les machines entre elles grâce à un switch.

## Routeur

Un routeur est un équipement matériel informatique dont la fonction principale consiste à orienter les données à travers un réseau. Chez nous, c’est notre box qui sert de routeur (entre autre).

## Le protocole TCP/IP

L’internet est constitué d’un ensemble de règles (protocole) qui permet à chaque ordinateur de communiquer avec d’autres.

Les données sont transportées sur le réseau grâce à plusieurs protocoles :

-   Le protocole **IP** (Internet Protocol) : permet de donner une adresse unique à un ordinateur sur le réseau, appelée adresse IP. Elle se présente sous la forme d’une série de 4 nombres (entre 0 et 255) séparés par des points, par exemple : 76.234.23.121 selon la norme IPV4. Ce protocole se charge de découper le message (textes, images, son, vidéos, etc.) en « paquets » et de les acheminer au destinataire mais sans garantie.
-   Le protocole **TCP** (Transmission Control Protocol) : S’appuie sur le protocole IP. Il s’assure que le transport des paquets est correctement assuré : il vérifie l’intégrité des données, il en garantit l’ordre et la remise, mais pas le délai de transmission.

## Les adresses IP

Une adresse IP est un numéro attribué à une machine selon l'Internet Protocole (IP). C'est habituellement (pour la version IPV4) une suite de 4 nombre pouvant aller de 0 à 255, séparés par des points.

#### Exemple :

```
197.168.56.24
```

### L'adresse réseau et l'addresse de la machine

Une adresse IP est composée de deux partie :

-   L'identification du réseau (souvent composée des 2 ou trois premiers nombres de mon adresse). Cette partie de l'adresse est aussi appelée "netID".
-   L'identification de la machine (souvent composée des 2 ou 1 dernier nombre de mon adresse). Cette partie de l'adresse est aussi appelée "HostID".

## Les paquets de données

Sur Internet, les données sont transférées sous forme de **paquets**. Chaque paquet est formé de deux grande partie :

-   **L'en-tête** où sont inscrites des informations permettant l'acheminement (adresse de l'expéditeur, du destinataire, ...
-   **La partie Data** qui contient une partie de la données à transférer.

## Les serveurs DNS

Le **serveur DNS** Domain Name System, ou Système de noms de domaine en français) est un service dont la principale fonction est de traduire un nom de domaine en adresse IP. Pour simplifier, le serveur DNS agit comme un annuaire que consulte un ordinateur au moment d'accéder à un autre ordinateur via un réseau.

Autrement dit, le serveur DNS est ce service qui permet d'associer à site web (ou un ordinateur connecté ou un serveur) une adresse IP, comme un annuaire téléphonique permet d'associer un numéro de téléphone à un nom d'abonné.
